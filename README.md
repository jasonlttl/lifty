# Lifty

A robo based command line utility for working with Acquia Lift.

## Quick Start

```
# Clone the repo.
git clone git@github.com/jasonlttl/lifty

# Enter the repo
cd lifty

# Install dependencies.
composer install

# Make a copy of the sample .env file and edit it with your credentials.
cp .env.sample .env

# See all commands.
robo list
```
